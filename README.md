# JS 台灣好食券適用店家地圖

已經綁定數位五倍券的民眾，記得從 10 月 8 日開始就可以帶著全家大小到貼有好食券標章的餐飲、糕餅、市場及夜市等小微店家與攤商，一起開心吃喝買。

## 使用方式
- 把整個專案複製一份到你的電腦裡，這裡指的「內容」不是只有檔案，而是指所有整個專案的歷史紀錄、分支、標籤等內容都會複製一份下來。
```sh
$ git clone
```
